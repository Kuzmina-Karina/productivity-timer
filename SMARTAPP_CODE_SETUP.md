## 🚀 Пошагово

### Шаг 1: Создать проект SmartApp Code

1. На https://developers.sber.ru/studio/
2. **Создать проект** → выбрать **"Code for SmartApp"**
3. Назвать **"Timer Script"** (или другое имя)
4. Создать

### Шаг 2: Загрузить сценарий в SmartApp Code

В SmartApp Code:

1. Откроется редактор
2. **Слева: "Экспорт/Импорт"**
3. Нажать **"Импортировать проект"**
4. Выбрать файл `scenario.zip` (ниже как создать)
5. **"Собрать"** (кнопка Build)

### Шаг 3: Создать scenario.zip

В терминале:

```bash
cd /Users/nastagetman/proga/todo-canvas-app/scenario

# Упаковать сценарий
zip -r ../scenario.zip . \
  -x "*.git*" \
  -x "test/*" \
  -x "node_modules/*"

cd ..
ls -lh scenario.zip
```

### Шаг 4: Получить webhook URL

В SmartApp Code (после сборки):

1. Меню слева → **"Публикации"**
2. Нажать **"Получить вебхук"**
3. Скопировать URL (выглядит как `https://dev.sber.ru/...`)

### Шаг 5: Создать Canvas App

1. На https://developers.sber.ru/studio/
2. **Создать проект** → выбрать **"Canvas App"** 
3. Назвать **"Timer App"** (или то же имя что и SmartApp Code)
4. **Группа** - та же что выбрали для SmartApp Code
5. Создать

### Шаг 6: Связать Canvas App со сценарием

В Canvas App (в новом проекте):

1. **Параметры** (gear icon)
2. **Сценарий смартаппа**
3. Выбрать тип: **"SmartApp Code API"**
4. **Внешняя ссылка:** вставить webhook URL из шага 4
5. **Frontend Endpoint:** `http://localhost:3000` (для тестирования)
6. **Сохранить**

### Шаг 7: Загрузить Canvas App

1. В Canvas App проекте → **Загрузить** (кнопка upload)
2. Выбрать из папки `/Users/nastagetman/proga/todo-canvas-app/build/`:
   - `index.html`
   - Папку `static/`
3. **Готово** → **Build**

### Шаг 8: Тестирование

В Canvas App:

1. Нажать **Play** (▶️ зеленая кнопка)
2. Говорить команды:
   - **"Запусти таймер на 30 секунд"**
   - **"Останови таймер"**
   - **"На 5 минут"**

## 🔧 Как это работает

### 1️⃣ Пользователь говорит голосовую команду

```
"Запусти таймер на 30 секунд"
```

### 2️⃣ SmartApp Code распознает и обрабатывает

- Сценарий парсит текст через `parseTimerDuration()`
- Отправляет команду Canvas App:
```javascript
{
  type: "smart_app_data",
  smart_app_data: {
    type: "set_timer",
    duration: 30
  }
}
```

### 3️⃣ Canvas App получает и обновляет UI

- React обновляет состояние таймера
- Timer компонент начинает отсчет
- Воспроизводится звуковой сигнал при завершении

### 4️⃣ При завершении Canvas App сообщает сценарию

```javascript
{
  action_id: "timer_finished"
}
```

### 5️⃣ SmartApp Code отправляет голосовой ответ

```
"Время вышло!"
```

## 🐛 Типичные ошибки

| Ошибка | Решение |
|--------|---------|
| "Интент не найден" | Перепроверьте intents.json и пересоберите SmartApp Code |
| "Canvas App не получает данные" | Убедитесь что webhook URL правильно добавлен в параметры |
| "Звук не работает" | Браузер блокирует аудио - разрешите в настройках |
| "Таймер не запускается" | Проверьте консоль (F12) на ошибки JavaScript |

---

## 📞 Если что-то не работает

1. **Откройте Console в браузере** (F12)
2. **Говорите команду** в Canvas App
3. **Ищите красные сообщения об ошибках**
4. **Проверьте Logs в SmartApp Code** (Menu → Logs)
