# ✅ ВСЁ ГОТОВО! Инструкция для SmartApp Code

## 📦 Что подготовлено

В папке проекта есть **два файла для загрузки**:

| Файл | Куда загружать | Назначение |
|------|----------------|-----------|
| **scenario.zip** | SmartApp Code | Сценарий с NLU интентами |
| **build/** или **timer-app.zip** | Canvas App | Веб-интерфейс таймера |

---

## 🚀 3 шага для запуска

### Шаг 1️⃣: Создать SmartApp Code

1. Откроить https://developers.sber.ru/studio/
2. **Создать проект** → **Code for SmartApp** (не Canvas App!)
3. Назвать **"Timer"**
4. Нажать **Создать**

### Шаг 2️⃣: Загрузить сценарий

В SmartApp Code:
1. **Экспорт/Импорт** (левое меню)
2. **Импортировать проект**
3. Выбрать файл **`scenario.zip`** из папки проекта
4. Нажать **Собрать** (Build)
5. Меню → **Публикации** → **Получить вебхук** (скопировать URL)

### Шаг 3️⃣: Создать Canvas App

1. На https://developers.sber.ru/studio/
2. **Создать проект** → **Canvas App**
3. Назвать **"Timer"** (же имя как SmartApp Code)
4. **Группа:** та же что выше
5. **Создать**

---

## ⚙️ Связать SmartApp Code с Canvas App

В Canvas App (новый проект):

1. **Параметры** (gear icon внизу)
2. **Сценарий смартаппа:**
   - Тип: **"SmartApp Code API"**
   - Внешняя ссылка: **вставить webhook из шага 2**
3. **Frontend Endpoint:** `http://localhost:3000`
4. **Сохранить**
