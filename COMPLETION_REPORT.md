### Для деплоя в SmartApp Studio:

1. **Подготовка:**
   ```bash
   npm run build
   # Файлы в папке build/ готовы
   ```

2. **Загрузка:**
   - Откройте SmartApp Studio
   - Создайте Canvas App проект
   - Загрузите build/index.html и build/static/

3. **Настройка:**
   - Импортируйте smartapp.json (интенты)
   - Установите REACT_APP_TOKEN
   - Протестируйте команды

4. **Публикация:**
   - Проверьте чек-лист в DEPLOY.md
   - Нажмите "Опубликовать"
